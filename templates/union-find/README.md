# union-find

Refer to the blog [Union Find in javascript](https://tkhwang.me/snippets/union-find-in-javascript)

### Files

- `unionFind.ts` – path compression + union-by-size implementation that reports component counts.
- `unionFind.test.ts` – Vitest suite covering unions, component sizes, and invalid inputs.

Run the specs from the templates root:

```bash
npm run test -- union-find
```
