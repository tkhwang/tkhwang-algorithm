# binary-search

Refer to the blog [binary search in javascript](https://tkhwang.me/snippets/binary-search-in-javascript)

### Files

- `binarySearch.ts` – implementation that exposes a typed `binarySearch` helper (numbers by default, custom comparator supported).
- `binarySearch.test.ts` – Vitest suite that drives the algorithm via TDD.

Run the specs from the templates root:

```bash
npm run test -- binary-search
```
